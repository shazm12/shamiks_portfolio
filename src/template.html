Movie Recommendation System

The Movie Recommendation System project aims to recommend 5 movies based on what movie the user had watched before.It uses Content-Based Recommendation Architecture where I used a movie dataset with names,genres,tags and such and then using Bag of Words technique, I derived cosine distance between the searched movie content which contained all the info from overview to keywords and even actors with all the other movie content present in the dataset and as an output, it gives 5 such movies with least cosine distance from all the content vectors present in the dataset.It was made using sklearn for modules like cosine similarity,Count Vectorizer and such and then made a simple web UI using Streamlit.
